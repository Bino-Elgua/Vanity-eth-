Option 1: Vercel (Fastest)

  $ npm run build
  $ git push dist/ to gh-pages branch

  $ netlify deploy --prod --dir=dist

  $ npm run build
  $ scp -r dist/* user@server:/var/www/html/vanity-eth-pro/
  (Any web server: nginx, Apache, Node.js, etc.)
