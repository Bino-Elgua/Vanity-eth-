THUNDER UPGRADE - ALL 5 FEATURES IMPLEMENTED
═══════════════════════════════════════════════════════════════════════════════

✅ 1. MULTI-CHAIN ENGINE (⛓️)
   • 6 blockchains: Ethereum, Solana, Bitcoin, Sui, Cosmos, Aptos
   • 3 crypto curves: secp256k1, ed25519, schnorr
   • 4 hash functions: keccak256, sha256, blake2b, sha3
   • Automatic address format per chain
   • QR code generation
   • Component: MultiChainGenerator.jsx
   • Utility: chainCrypto.js

✅ 2. HD WALLET GENERATOR (🌱)
   • BIP-39 mnemonic (24-word, 256-bit entropy)
   • BIP-32/BIP-44 hierarchical derivation
   • Multi-chain wallet derivation from single seed
   • Standard derivation paths (m/44'/coin'/0'/0)
   • Import/export seed phrases
   • Component: HDWallet.jsx
   • Utility: hdWallet.js

✅ 3. POISON RADAR (📡)
   • On-chain safety scanner (public RPC only)
   • Dust detection, zero-value flagging, spam patterns
   • Risk scoring: LOW / MEDIUM / HIGH
   • 100% client-side, no servers
   • Component: PoisonRadar.jsx
   • Utility: poisonRadar.js

✅ 4. ZERO-KNOWLEDGE EXPORT (🔐)
   • AES-256-GCM encryption
   • PBKDF2 key derivation (100k iterations)
   • 3 export modes:
     1. Password-locked keystore (MetaMask compatible)
     2. One-time QR codes (TTL-based burn)
     3. Verified Base64 + SHA-256 checksum
   • Component: ExportManager.jsx
   • Utility: encryption.js

✅ 5. BATCH & PROFILE MODE (🔄)
   • Create/edit/delete profiles
   • Batch job execution (queue multiple searches)
   • localStorage persistence (no server)
   • Export/import profiles as JSON
   • Profile statistics & tracking
   • Component: ProfileManager.jsx
   • Utility: profiles.js

HOW TO USE
═══════════════════════════════════════════════════════════════════════════════

Development Server:
  $ cd vanity-eth-pro
  $ npm install
  $ npm run dev
  # Opens http://localhost:5173

Production Build:
  $ npm run build
  $ npm run preview
  # Opens http://localhost:4173

Deploy to Vercel (Recommended):
  $ npm i -g vercel
  $ vercel
  # Auto-deployed in 1 minute

Deploy to GitHub Pages:
  $ npm run build
  $ git push dist/ to gh-pages branch

Deploy Self-Hosted:
  $ npm run build
  $ scp -r dist/* user@server:/var/www/vanity-eth-pro/
